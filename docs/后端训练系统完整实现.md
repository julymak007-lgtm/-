## 🧠 后端训练功能详解

### 1. LSTM模型训练 (`train_lstm.py`)

#### 模型架构
```python
# 完全按照PRD实现的LSTM架构
Sequential([
    LSTM(64, return_sequences=True, input_shape=(240, 20)),  # 4小时 = 240分钟
    Dropout(0.2),
    LSTM(32),
    Dropout(0.2),
    Dense(16, activation='relu'),
    Dense(1, activation='sigmoid')
])
```

#### 特征工程
- **原始特征**: 真空度、气压、抛料率等30+参数
- **统计特征**: 滑动窗口均值、标准差、最大值、最小值、斜率
- **领域特征**: CPH效率、热应力标志、单位抛料率

#### 训练流程
```
1. 连接数据库 → 2. 加载标注数据 → 3. 提取特征
   ↓
4. 标准化特征 → 5. 构建LSTM序列 → 6. 模型训练
   ↓
7. 模型评估 → 8. 导出ONNX格式 → 9. 保存结果
```

### 2. 生存分析模型训练 (`train_survival.py`)

#### 支持的模型
- **Weibull AFT**: 适合有明确分布的数据
- **Cox PH**: 半参数模型，适用性广  
- **Random Survival Forest**: 非线性关系建模

#### 输出
- 剩余使用寿命 (RUL) 预测
- 生存函数曲线
- C-index 评估指标

### 3. 任务队列系统 (`task_queue.py`)

#### 功能特性
- ✅ 异步任务提交
- ✅ 任务状态追踪
- ✅ Worker进程管理
- ✅ 结果持久化
- ✅ 数据库自动更新

#### 使用方式

```bash
# 启动训练Worker
python backend/training/task_queue.py

# 或使用Celery (需要Redis)
celery -A task_queue worker --loglevel=info
```
